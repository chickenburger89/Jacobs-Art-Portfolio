About

This website was the first one we created for my HTML web design class. The assignment emphasized fundamental skills such as properly using CSS, choosing appropriate fonts and colors, and producing a layout that makes the user experience smooth and easy. I enjoyed working on this project and I learned a lot doing so. This was the first major assignment we had to work on, and it built upon the skills we had been learning up to that point in the class. One challenge I faced was making the gallery section, as it required multiple pages that are carefully interlinked. Although it was not technically required, I wanted to do so to challenge myself and learn a little bit more about HTML.
