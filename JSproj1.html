P5 Pokédex JavaScript Project

About

This is a JavaScript data visualization project using the P5 library. The goal of this assignment was to utilize JSON data to create a visualization of some kind. When browsing the different datasets to use for the project, I found the Pokemon one and decided to create something around that. It was easy to make a program that sorts and prints the list of Pokemon, but where I really wanted to challenge myself was making the image of the Pokemon appear and the sound of its cry play when you click on the names. I have loved Pokemon since I was a kid, and I wanted to create a project that reflected that. Even though I struggled a bit, I was committed to creating a project I am very proud of.

DISCLAIMER: This project contains images and sounds that are not mine. I do not take credit for them. It was made purely as a demonstration of my programming skills.
